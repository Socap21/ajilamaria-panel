# Capturas de pantalla

Coloca aquí las imágenes que el `README.md` muestra. Usa exactamente estos nombres:

- `login.png` — la pantalla de inicio de sesión (con el título en llamas).
- `dashboard.png` — el panel de resumen (KPIs y gráficas).
- `productos.png` — *(opcional)* la vista de productos e inventario.
- `reportes.png` — *(opcional)* la vista de reportes.
